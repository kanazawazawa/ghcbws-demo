# Excercise2
## プログラムの実行
Exercise2 のディレクトリに移動
```bash
cd Exercise2
```
npm installを実行
```bash
npm install
```
npm startを実行
```bash
npm start
```
ブラウザで http://localhost:3000/ にアクセス
Ctrl + Cで終了
